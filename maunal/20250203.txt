  Chapter07 : 인라인과 블록 및 박스 모델 관련 수업

    1. 인라인 요소 / 블록 요소 : 
      CSS 파트에서 가장 중요한 부분
        -> 웹 페이지의 공간을 구성하는 기본 방식에 해당, 두 개념 확실히 알고
        넘어가야 웹 사이트 만드는 과정 수월해짐

      1) 기본 너비와 개념 : 
        HTML 의 경우 기능이 포함된 태그가 많으나 span 태그 div 태그의 경우 기능
        그 자체가 있는것이 아닌 구획을 나눠주는 역할 함

        해당 태그들의 차이는 블록 요소인지 인라인 요소인지에 대한 차이

        n1inlineblock01생성

        inline vs block 

        inline : 글자가 차지하는 만큼의 너비를 의미
        block : 한 줄을 기준으로 전체를 채우는 역할

      n0example

      display: inline-block --> 기본적으로 인라인 요소의 속성을 가지며 너비와 높이 조절이 가능.

      2) 너비와 높이 조절
        : width 속성 
          height 속성 -> n0example 에서 확인

          inline / block / inline-block 의 성격상 차이  
          inline : span 태그와 같이 글자가 차지하는 범위만큼의 구역
          block : 글자가 있는 브라우저 상의 가로줄 : 너비/ 높이 조절 가능
          inline-block : 기본적으로 글자가 차지하는 범위만큼 구역이지만
            너비 / 높이 조절 가능

      3) 가로 공간
        width 지정하지 않을 경우 인라인 요소는 컨텐츠 너비 만큼만,
        블록 요소는 줄 전체 공간 차지한다는 것을 확인 가능.

        가로 공간을 전체 차지하는 블록 요소를 inline-block으로 지정시 예시

      4) 여백적용 : 
        margin : 컨텐츠 요소의 바깥쪽 여백 지정
        padding : 컨탠츠 요소의 안쪽 여백 지정

  2. 박스모델 : 

    1) 요소의 크기 :  
      width / height 속성을 통해 block / inline-block의 경우 크기를 조절 가능
      -> 고정값 크기의 경우 px 사용.
      --> 웹사이트는 브라우저나 디바이스에 따라 화면 크기가 유동적으로 변함
      따라서 어떤 요소의 크기를 고정해버리면 가독성 떨어지는 문제 생길수 있음
      ---> 이상의 문제 해결을 위해 %를 사용해 부모의 크기에 비례한 값을 가지게 할 수있음

      ex) width 속성을 100% 로 줬다면 부모 크기에 맞는 크기가 되기에 창의 크기를 어떻게 
      조절하더라도 너비가 꽉찬 상태 유지함

      n3aexample

      결과 디바이스에 따라서 유동적으로 크기를 조절 할 때 유용한 기능 :
        -> 뷰포트(viewport) : '브라우저의 크기'

         vw (viewport width) : 뷰포트 기준으로 했을 때의 너비가
         vh (viewport height) : 뷰포트 기준으로 했을 때의 높이
         vmax (viewport max) : 상대적 길이가 긴쪽
         vmin (viewport min) : 상대적 길이가 짧은쪽
         
            라는 단위 존재 -> 1/100 으로 나누어왔음
         3,4의 경우 컴퓨터 모니터의 경우 가로길이 > 세로길이 인데 반해
         모바일의 경우 가로길이<세로길이 와 같은 방식으로 유동적으로 창의 크기가 변하는 경우
         사용

         calc() 함수 : 매개변수를 이용 CSS 에서 특정한 수치를 계산하는 함수
          ex) width에 calc(100% +100px)을 넣는다면
          상대값 100% 수치에 100px 더 많도록 나타내는 방식

          vw,vh에도 적용 가능
          padding 이 적용 되어있을 때 사용
          ex) 요소 안쪽 24px 의 padding 이 잡혀있다면 원하는 상대값에서 24px 을 두번 뺀
          calc(100%-48px) 로 넣어야 원하는 길이 측정 가능

      바깥쪽 여백 // 안쪽 여백

        margin 속성 :
          마진 지정 방식
            (1) margin 값을 하나로 지정하는 방법 :
              기본적으로 여백은 모든부분에 공통적으로 적용
              ex) margin: 24px 로 적용 시 
              컨텐트 상하좌우 모든 방향에 24px 의 margin 영역이 발생
              em 지정의 경우도 동일
              0의 경우 단위 지정 없어도 됨

            (2) margin 값 두개로 지정하는 방법 :  
              첫번째 값 : 컨텐츠 세로영역
              두번째 값 : 컨텐츠 가로영역
